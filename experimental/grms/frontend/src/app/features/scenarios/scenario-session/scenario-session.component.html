Live Scenario Simulation

Run real-time geopolitical scenarios by connecting to a WebSocket session. Select a leader profile and country, then inject events to see how the leader and population would respond.

  • Scenario ID — a unique session name (any string). Multiple users with the same ID share a session.
  • Leader — the decision-maker whose cognitive profile drives responses. Profiles include personality, ideology, and decision history.
  • Country — determines which population model reacts to the leader's actions.
  • Events — geopolitical stimuli (military, diplomatic, economic, cyber, information) with severity and reversibility. Compose events in the bottom bar.

Responses are generated by the LLM prediction engine using the selected leader's full profile. The timeline shows both leader actions and population sentiment shifts.
