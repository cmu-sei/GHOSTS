Cognitive Decision Engine

This lab lets you explore how a leader's decision emerges from six cognitive layers, each contributing to a final action score. Adjust any parameter and see the decision shift in real time.

  • Identity — role constraints and decision-making style. Heads of state with advisory councils resist unilateral extremes.
  • Personality — 8 dimensions (risk tolerance, aggression, nationalism, etc.) that bias toward escalatory or restrained actions.
  • Beliefs — Bayesian posteriors about the world (is the threat real? would military action work?). These amplify or suppress action options.
  • Motivations — underlying drives (power, honor, independence, vengeance). Actions that satisfy core motivations score higher.
  • Context — situational pressures (approval rating, military readiness, red line status). High approval + red line crossed = permission for strong action.
  • History — past decisions create consistency biases. Leaders tend to repeat what worked before.

The right panel shows layer weights (adjustable) and sensitivity analysis — how the decision changes as you sweep a single parameter across its range.

This is a parametric model with no LLM dependency. It runs instantly and produces a fully auditable decision trace.
